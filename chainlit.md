# Bedrock Chat

A private chat interface for AWS Bedrock models. Your conversations are stored
locally on this machine; the only data that leaves it are the API calls to
your own AWS account.

- Pick a model in the **settings panel** (⚙️ icon next to the message box).
- Models tagged `[img]` accept image uploads; `[doc]` accept documents
  (PDF, Word, Excel, CSV, TXT, MD, HTML).
- Attach files with the paperclip or by dragging them into the chat.
- Past conversations are in the sidebar and can be resumed.
